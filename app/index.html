About me

I'm a web developer based in Germany, currently living near Berlin. I'm a student of Environmental Management, but technology was always my main interest.

I spend most of my free time tirelessly learning and working on projects. I'm looking for freelance projects at the time, so don't hesitate to message me if you have something in mind.

My skills

I'm proficient with HTML and CSS. I code responsive websites, with or without the help of Bootstrap. I write my CSS with the help of Sass and use Gulp as my build tool. I make my projects interactive with Javascript and jQuery. I'm proficient with version control and Git, and I used it to collaborate on group projects before. When a project reqires more than a static site, I use WordPress as my CMS of choice.

Learning

My current goal is to learn React and build rich web applications I'm also planning to add testing to my Javascript skillset. I plan to learn Node.js and Express as well, but full stack development is not my main goal right now. Incorporating Material Design into my projects is another potential interest.
