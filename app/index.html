About me

I'm a web developer based in Germany, currently living near Berlin. I'm a student of Environmental Management, but technology was always my main interest.

These days, I spend most of my free time tirelessly learning and working on projects.

My skills

I'm proficient with HTML and CSS. I code responsive websites, with or without the help of Bootstrap. I write my CSS with the help of Sass and use Gulp as my build tool. I make my projects interactive with Javascript and jQuery. I'm proficient with version control and Git, and I used it to collaborate on group projects before.

Learning

Recently I've been polishing my CSS skills, but my current goal is to learn React. I'm learning ES6 features before I make that big step. I'm also planning to add testing to my skillset, and I'm considering learning Wordpress as well. I plan to learn Node.js and Express as well, but full stack development is not my main goal right now. Incorporating Material Design into my projects is another potential interest.
